1. LE BUT DU JEU

Pixel War V4 est une expérience sociale massivement multijoueur. Vous êtes sur une toile géante partagée par tout le monde.

  • Chaque joueur peut poser 1 pixel toutes les 30 secondes.
  • Coordonnez-vous pour créer des œuvres d'art.
  • Protégez vos créations contre les attaques ennemies.

2. SYSTÈME DE GUILDES

Ne jouez pas seul ! Les guildes dominent la carte.

Utilisez le menu "GUILDES" pour rejoindre une équipe. Vous aurez accès à un chat privé et pourrez coordonner vos attaques pour grimper dans le classement.

3. CODE DE CONDUITE

Pour garder la communauté saine :

  • Pas de dessins haineux, racistes ou sexuellement explicites (Ban immédiat).
  • L'utilisation de scripts ou de bots automatisés est interdite.
  • Respectez les autres joueurs dans le chat.
